# Política de Agentes

1. Siempre coordinar desde el Coordinator.
2. Invocar solo expertos necesarios.
3. Compartir únicamente el contexto mínimo requerido.
4. Registrar decisiones importantes en la memoria.
5. No duplicar responsabilidades.
6. Si el stack o dominio lo requiere, crear agentes dinámicos (React, .NET, Viajes, eCommerce, etc.).
