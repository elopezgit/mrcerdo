# PROJECT PROFILE

- **Nombre:** Suplementos AR
- **Dominio:** eCommerce / Suplementos Deportivos
- **Objetivo:** Sistema multi-tenant de catálogo digital y pedidos para tiendas de suplementos (proteínas, creatinas, pre-entrenos, quemadores, vitaminas)
- **Stack:** React 19 + TypeScript 6 + Tailwind CSS 4 + Vite 8 + Supabase
- **Arquitectura:** SPA multi-tenant con rutas por slug (`/:slug`, `/admin/:slug`)
- **Estado:** En desarrollo activo
- **Módulos:** Catálogo público, POS, Panel admin (kanban, analíticas, gestión)
- **Riesgos:** RLS de Supabase bloqueando acceso público
- **Integraciones:** Supabase Auth + PostgreSQL, WhatsApp (deep link)
