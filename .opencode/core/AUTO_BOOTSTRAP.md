# Auto Bootstrap

Objetivo: inicializar automáticamente cualquier proyecto.

## Flujo

1. Detectar si el proyecto es nuevo o existente.
2. Ejecutar Discovery.
3. Generar PROJECT_PROFILE.
4. Generar PROJECT_MEMORY.
5. Detectar stack.
6. Detectar dominio.
7. Detectar arquitectura.
8. Crear agentes dinámicos.
9. Crear skills dinámicas.
10. Generar plan.
11. Esperar aprobación.
12. Comenzar implementación.
