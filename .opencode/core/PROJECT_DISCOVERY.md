# Project Discovery

Objetivo: comprender completamente un proyecto antes de modificarlo.

## Etapas
1. Detectar tipo de proyecto.
2. Detectar stack.
3. Detectar dominio.
4. Detectar arquitectura.
5. Detectar módulos.
6. Detectar dependencias.
7. Detectar integraciones.
8. Detectar riesgos.
9. Detectar deuda técnica.
10. Generar resumen.
