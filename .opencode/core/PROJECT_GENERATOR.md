# Dynamic Project Generator

## Objetivo
Generar automáticamente la estructura IA para cualquier proyecto.

## Flujo
1. Detectar proyecto nuevo o existente.
2. Ejecutar Discovery.
3. Clasificar dominio y stack.
4. Generar memoria inicial.
5. Generar agentes necesarios.
6. Generar skills necesarias.
7. Generar documentación.
8. Generar roadmap inicial.
9. Esperar aprobación.
10. Iniciar desarrollo.
