# Plan de Mejora Continua

## Objetivo
Identificar y ejecutar acciones de mejora basadas en métricas de calidad y retrospectivas.

## Formato de acción de mejora
```
## Mejora #[ID]
- **Problema**: [descripción]
- **Causa raíz**: [análisis]
- **Acción propuesta**: [qué hacer]
- **Dueño**: [responsable]
- **Fecha límite**: [fecha]
- **Métrica de éxito**: [cómo medir]
- **Estado**: [Pendiente / En curso / Completada / Cancelada]
```

## Prioridades
| Prioridad | Criterio | Tiempo de respuesta |
|---|---|---|
| Crítica | Bloquea entregas o afecta producción | Inmediato |
| Alta | Deuda técnica creciente o calidad degradándose | Este sprint |
| Media | Mejora de proceso o prevención | Próximo sprint |
| Baja | Optimización o refactorización menor | Backlog |

## Ciclo de mejora
```
Medir → Identificar → Planificar → Ejecutar → Validar → Estandarizar
```
