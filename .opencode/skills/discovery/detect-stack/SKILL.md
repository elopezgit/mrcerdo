# detect-stack

## Objetivo
Detect Stack.

## Cuándo usar
El Coordinador debe invocar esta Skill únicamente cuando sea necesaria para resolver la tarea.

## Entradas
- Contexto resumido
- Archivos relevantes
- Memoria del proyecto

## Procedimiento
1. Analizar únicamente el contexto necesario.
2. Evitar releer archivos ya comprendidos.
3. Generar un resultado breve y accionable.
4. Actualizar la memoria si corresponde.

## Salida
Resumen, hallazgos, riesgos y próximos pasos.
